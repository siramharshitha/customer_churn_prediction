# Customer Churn Prediction Using Machine Learning

## Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company's service. The objective is to help businesses identify at-risk customers and take proactive measures to improve customer retention and reduce revenue loss.

## Features

* Data Cleaning and Preprocessing
* Handling Missing Values
* Categorical Data Encoding
* Exploratory Data Analysis (EDA)
* Random Forest Classification Model
* Model Evaluation and Performance Metrics
* Confusion Matrix Visualization
* Feature Importance Analysis
* Customer Churn Prediction

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset

The project uses the Telco Customer Churn Dataset, which contains customer demographic information, account details, service subscriptions, billing information, and churn status.

## Project Workflow

1. Load and preprocess the dataset.
2. Handle missing and inconsistent data.
3. Convert categorical features into numerical format.
4. Split data into training and testing sets.
5. Train a Random Forest Classifier.
6. Evaluate model performance using Accuracy, Precision, Recall, and F1-Score.
7. Visualize results with Confusion Matrix and Feature Importance charts.
8. Predict customer churn based on customer information.

## Results

The model successfully identifies customers who are likely to leave the service, helping organizations improve customer retention strategies and business decision-making.
